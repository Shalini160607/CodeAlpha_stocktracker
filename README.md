# CodeAlpha Stock Portfolio Tracker

## Description
A Python program that calculates total investment value using predefined stock prices and user-entered quantities.

## Features
- Displays available stocks
- Takes stock name and quantity
- Calculates total investment
- Saves result to a text file

## Technologies Used
- Python

## Concepts Used
- Dictionary
- Input/Output
- Arithmetic operations
- File handling

## How to Run

python stock_tracker.py
